# IPL Data Analysis Dashboard (2008–2025)

## Overview

Interactive Excel dashboard built using IPL data from 2008 to 2025.

## Features

- Dynamic Season Slicer
- Winner Analysis
- Runner-Up Analysis
- Player of the Season Analysis
- Team-wise Wins Analysis
- Toss Decision Analysis
- Top 10 Venues Analysis
  
## Skills Demonstrated
 
- Data Cleaning
- Data Analysis
- Data Visualization
- Dashboard Design
- Pivot Tables
- Pivot Charts
- Slicers
- VLOOKUP

## Business Insights

- Most successful IPL teams
- Toss decision trends
- Venue-wise match analysis
- Season-wise performance analysis
- Player of the Season tracking

## Tools Used

- Excel / WPS Office
- Pivot Tables
- Pivot Charts
- Slicers
- VLOOKUP
